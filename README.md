# AWS VPC Web Server Project

This project demonstrates how to create a custom AWS VPC with:
- Public and private subnets  
- Route tables  
- Internet Gateway  
- Security Groups  
- EC2 instance deployed via SSH  
- Web server running Nginx  

## Technologies Used
- Amazon VPC  
- Subnets  
- Route Tables  
- Internet Gateway (IGW)  
- EC2 (Amazon Linux 2023)  
- SSH  
- Nginx Web Server  

## Project Steps
1. Created a custom VPC  
2. Added public and private subnets  
3. Attached an Internet Gateway  
4. Configured route tables and associations  
5. Launched an EC2 instance  
6. Connected via SSH using a key pair  
7. Installed and configured Nginx  
8. Hosted a custom webpage
